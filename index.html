ক্যামেরা চালু হয়নি

Camera access পাওয়া যায়নি।

  • OBS Browser Source হলে Local File এর বদলে http://localhost URL ব্যবহার করুন
  • OBS চালুর সময় --use-fake-ui-for-media-stream flag ব্যবহার করুন
  • Windows/Mac privacy settings থেকে OBS-কে Camera permission দিন
  • অথবা ফাইলটি Chrome-এ খুলে Window Capture দিয়ে OBS-এ নিন
